# Company Logos

Place company logo images in this folder with the following names:

## Required Logo Files:
- `amazon.png` - Amazon logo (white version for visibility on blue background)
- `google.png` - Google logo
- `dribbble.png` - Dribbble logo (white version)
- `x.png` - X (Twitter) logo (white version)
- `airbnb.png` - Airbnb logo (white version)
- `apple.png` - Apple logo (white version)

## Image Specifications:
- Format: PNG with transparent background
- Size: Recommended 128x128px or larger (will be scaled to 32x32px)
- Color: White logos work best on the blue sphere background
- The logos will be displayed inside circular blue spheres with 3D effects

## Fallback:
If a logo file is not found, the first letter of the company name will be displayed as a fallback.
